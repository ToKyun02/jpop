## 프로젝트 목적

| 국내 사용자에게 최적화 된 jpop 정보 사이트

## 프로젝트 내용 상세

1. 최근 인기있는 jpop 곡과 가수들의 정보를 serving
2. ai prompt를 활용한 playlist 자동 생성
3. 다양한 cover 곡 serving
4. jpop 내한 일정을 크롤링하여 정보 serving

그 외 기능 확장

## tech stack

### front

react19, vite, mantaine, ts, react-router

### back

django(향후 spring boot로 전환), MySQL, Elastic Search

### Infra

EC2, Docker(선택적), Nginx

### report

Sentry
